项目总体综述
本项目是一个基于 Java 语言开发的 B/S 架构在线购物平台。系统采用最新的 Spring Boot 3 框架进行构建，结合 MySQL 数据库实现数据的持久化管理。项目涵盖了从用户身份认证、商品展示、订单处理到销售数据可视化的完整业务闭环。
核心功能实现逻辑
1. 顾客端功能模块
身份认证系统：实现用户注册、登录、注销功能，利用 HttpSession 机制维护用户会话状态。
响应式商品展示：首页采用网格化布局展示数码产品卡片，根据设备屏幕尺寸自动适配显示效果。
自动化购买流程：
事务控制：通过 @Transactional 注解确保库存扣减与订单生成的原子性，防止数据异常。
邮件通知：集成 SMTP 协议，在订单确认后自动向用户注册邮箱发送发货通知。
订单追踪：用户可实时查看个人名下的历史订单信息与当前发货状态。
2. 销售管理模块
访问控制：通过自定义拦截器对管理路径进行权限校验，非管理员账号无法进入。
目录维护：提供商品信息的增加、删除、修改、查询（CRUD）功能，支持实时调整库存。
数据可视化报表：利用 ECharts 图表库，通过后端 SQL 聚合查询获取真实销售额，生成动态统计折线图。
